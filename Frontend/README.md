
# Deepfake Detection Frontend

This is the React frontend for the Deepfake Video Detection project. It provides the user interface for uploading files, sending them to the backend, and displaying the analysis result.

## Features

- Upload video file
- Upload reference image
- Send data to backend API
- Show analysis result
- Display:
  - prediction label
  - score
  - threshold
  - processed frames
  - detected frames
  - analysis graph

## Tech Stack

- React
- JavaScript
- HTML
- CSS
# Prerequisites

Make sure you have:
```
Node.js installed
npm installed
```
Step 1 : Go to the frontend folder
```bash
cd Frontend
```
Step 2 : Install dependencies
```bash
npm install
```
Step 3 : Run the Frontend

```bash
npm start
```
